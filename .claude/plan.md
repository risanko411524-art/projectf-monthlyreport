# 月報アプリ アップデート計画

## 変更点

### 1. カラーテーマ変更（バナナグループ）
- 紫/赤系 → 緑/黄色のグラデーション
- 背景を深緑ベースのダークテーマに
- アクセントカラーを黄色・ライムグリーンに
- 背景のorbアニメーションも緑/黄色系に

### 2. バナナキャラクター追加
- CSSアニメーション付きのバナナキャラをヘッダーやスタート画面に配置
- 各ステップ完了時にバナナが揺れるなどの演出

### 3. Google Spreadsheet連携
- Google Apps Script (GAS) でWebアプリを作成 → そのURLをHTMLに設定
- 提出時に `fetch()` でGASのURLにPOST
- GAS側でスプレッドシートに各項目を1行として追記
- セットアップ手順をコメントで記載

### 4. Slack用コピー機能
- 提出完了画面に「Slack用にコピー」ボタン追加
- フォーマット済みテキスト（絵文字付き）を生成
- クリックでクリップボードにコピー
- 各ステップ入力後にもまとめプレビューを表示

### 5. スプレッドシート出力内容
- 各項目（Instagram数値、売上、7つの質問の回答）を個別カラムで出力
- 最後にSlack用のまとめテキストも1カラムとして出力

## 対象ファイル
- `index.html` — 全面書き換え
- `README_SETUP.md` — GASのセットアップ手順（新規作成）
